# Office on Zorin OS Using Play On Linux

This repository contains scripts and tools to install Microsoft Office on Zorin OS (and other Linux distributions) using Wine and PlayOnLinux.

The project is split into two parts:

*   **[Office 2016](./office16/README.md)**: Scripts and instructions for installing the offline version of Microsoft Office 2016.
*   **[Office 365](./office365/README.md)**: An automated installer for Office 365.

Please navigate to the respective directories for detailed instructions.
